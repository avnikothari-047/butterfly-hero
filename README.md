# Butterfly Hero Animation Assignment

Interactive hero section built using Next.js, Tailwind CSS, and GSAP.

A draggable butterfly moves horizontally and smoothly reveals the headline and statistics based on user interaction. The project focuses on motion quality, easing, responsiveness, and clean animation logic.

## Tech Stack
- Next.js
- Tailwind CSS
- GSAP

## Run Locally
npm install
npm run dev
